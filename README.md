### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate environment

```bash

```

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install fastapi uvicorn
```

### 5. Run the server

```bash
uvicorn main:app --reload
```

Open in browser:

* API: http://127.0.0.1:8000
* Docs: http://127.0.0.1:8000/docs
